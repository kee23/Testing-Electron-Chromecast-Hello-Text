# Testing-Electron-Chromecast-Hello-Text

**Clone and run for a quick way to see Electron with Chromecast in action.**

This is a minimal Electron application based on [Electron Quick Start](https://github.com/electron/electron-quick-start).

This app also uses [Electron-Chromecast](https://github.com/GPMDP/electron-chromecast).

## To Use

```bash
# Clone this repository
git clone https://github.com/kee23/Testing-Electron-Chromecast-Hello-Text.git
# Go into the repository
cd Testing-Electron-Chromecast-Hello-Text
# Install dependencies
npm install
# Run the app
npm start
```
